## Технические детали

### Новые компоненты

1. **MultiSelect** (`app/components/ui/MultiSelect.vue`)
   - Props: `modelValue`, `options`, `label`, `placeholder`, `hint`, `searchable`, `required`
   - Events: `update:modelValue`
   - Особенности: поиск, множественный выбор, темная тема, анимации

### API endpoints (уже существующие)

- `GET /api/courses` - список программ с пагинацией
- `GET /api/courses/:id` - получение программы по ID
- `POST /api/courses` - создание программы
- `PUT /api/courses/:id` - обновление программы
- `DELETE /api/courses/:id` - удаление программы
- `GET /api/courses/templates` - список шаблонов сертификатов
- `GET /api/instructors/all` - список всех инструкторов

### Маршруты

- `/programs` - список учебных программ
- `/programs/create` - создание новой программы
- `/programs/:id` - просмотр программы

## Стилистика и UX

### Цветовая схема

- Primary - основные элементы
- Success - активные программы, инструкторы
- Warning - часы, время
- Danger - удаление, неактивные элементы

### Анимации

- Плавные переходы при наведении
- Анимация появления/исчезновения уведомлений
- Анимация открытия/закрытия выпадающих списков
- Анимация модальных окон

### Адаптивность

- Мобильная версия (sm, md)
- Планшетная версия (lg)
- Десктопная версия (xl, 2xl)
