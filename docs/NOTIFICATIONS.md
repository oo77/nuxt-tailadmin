# Система уведомлений (Notifications)

## Описание

Система уведомлений предоставляет элегантный способ показа сообщений пользователю с анимацией и автоматическим закрытием.

## Использование

### Базовое использование

```typescript
const notification = useNotification();

// Успешное уведомление
notification.success("Операция выполнена успешно", "Успех");

// Ошибка
notification.error("Произошла ошибка", "Ошибка");

// Предупреждение
notification.warning("Внимание! Проверьте данные", "Предупреждение");

// Информация
notification.info("Новая информация доступна", "Информация");
```

### Параметры

- `message` (обязательный) - текст сообщения
- `title` (опциональный) - заголовок уведомления
- `duration` (опциональный) - время отображения в миллисекундах (по умолчанию 5000мс, 0 = не закрывать автоматически)

### Примеры

```typescript
// С пользовательским временем отображения
notification.success("Данные сохранены", "Успех", 3000);

// Без автозакрытия
notification.error("Критическая ошибка", "Ошибка", 0);

// Закрыть все уведомления
notification.closeAll();

// Получить функцию закрытия конкретного уведомления
const { close } = notification.info("Загрузка...", "Процесс");
// Позже можно закрыть вручную
close();
```

## Типы уведомлений

- **success** - зеленый цвет, для успешных операций
- **error** - красный цвет, для ошибок
- **warning** - желтый цвет, для предупреждений
- **info** - синий цвет, для информационных сообщений

## Особенности

- ✅ Анимация появления/исчезновения
- ✅ Автоматическое закрытие с прогресс-баром
- ✅ Поддержка темной темы
- ✅ Возможность закрытия вручную
- ✅ Множественные уведомления (стек)
- ✅ TypeScript поддержка

## Интеграция в формы

Пример использования в валидации формы:

```typescript
const handleSubmit = async () => {
  const notification = useNotification();

  // Валидация email
  if (!validateEmail(formData.email)) {
    notification.error("Введите корректный email адрес", "Ошибка валидации");
    return;
  }

  // Валидация пароля
  if (formData.password.length < 6) {
    notification.error(
      "Пароль должен содержать минимум 6 символов",
      "Ошибка валидации"
    );
    return;
  }

  try {
    await saveData();
    notification.success("Данные успешно сохранены", "Успех");
  } catch (error) {
    notification.error("Не удалось сохранить данные", "Ошибка");
  }
};
```

## Компоненты

### useNotification (composable)

Основной composable для работы с уведомлениями.

**Файл:** `app/composables/useNotification.ts`

### Notification (компонент)

UI компонент для отображения уведомления.

**Файл:** `app/components/ui/Notification.vue`

## Стилизация

Уведомления автоматически адаптируются под светлую и темную темы. Цветовая схема определяется типом уведомления.
